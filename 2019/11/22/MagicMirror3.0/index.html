自动安装

失败率很高，建议手动安装
在终端直接输入bash -c "$(curl -sL https://raw.githubusercontent.com/MichMich/MagicMirror/master/installers/raspberry.sh)"耐心等待，将会安装所有内容，安装完毕后在终端输入cd MagicMirror,npm start,如果出现报错即未安装成功

手动安装

打开树莓派终端或者SSH连接树莓派，在终端中依次执行以下命令

到此为止上面为安装准备工作，务必保证以上环节没有报错再进行下一步
官方的方法是用npm install,但是npm的服务器在国外，在国内的速度及其之慢，500M的带宽也是小水流的速度，所以这里改成cnpm,cnpm是淘宝的镜像源，10分钟同步一次放在国内的服务器上，速度成倍增加。
安装cnpm：在终端输入sudo npm install -g cnpm

安装完成之后，确保你在MagicMirror目录下，输入cnpm install,稍许等待。


至此为止就完成了安装。

启动MagicMirror

在终端输入npm start,如果出现如下提示，则安装成功

不用担心，这是因为系统给的config文件名后缀多了一个“sample”，去掉即可，或者重新在目录下新建config.js，将内容复制进去就行，config.js.sample文件可用来备份，万一config修改错了可以还原。
文件所在位置：/home/pi/MagicMirror/config,重命名两种方式，一种是图形化界面右击rename，此方法不再赘述，另一种没有GUI时，终端进入config目录cd /home/pi/MagicMirror/config,mv config.js.sample config.jstips:Linux中按tab可自动补全命令，然后再返回上级目录，执行npm start,就可以看到已经成功啦！
