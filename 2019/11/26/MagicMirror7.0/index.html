MagicMirror系列教程(七):硬件选购与安装
Created 2019-11-26| Updated 2019-11-29|MagicMirror
Post View:

  在开始之前请浏览教程首页，以便访问常用链接以及总目录。

  硬件选择与安装部分，好多人有问题，纠结不知如何选购，本篇简单介绍，可作为挑选的参考，但DIY的精髓所在即自己动手尝试，做决定胆大一点心细一点，不要再让别人推荐屏幕呀木框呀啥的了，这样的作品么得灵魂。

树莓派主机

  实现MagicMirror效果，小白推荐使用树莓派,因为使用用户多，相关资料也比较多，在使用过程中，遇到问题大概率其他人也遇到过，方便交流。推荐使用2b,2b+,3b,3b+其中某款，4代的目前价格偏高，如有其它需求可选购4代，如果只为这个项目，买一个2手3代即可，能省则省。

版本选择：RS版和E14版均可，只是英国版和国产版而已，不要太多纠结。
套餐选择：
必备：主板，电源线，8G以上内存卡，读卡器
可选：散热片，风扇外壳，（传感器套件按需）
各种连接线：根据屏幕的接口，选择树莓派与屏幕的连接线，通常为HDMI线、HDMI转VGA线

显示屏

  因为是做镜子，所以3.5寸那种强烈不推荐，7寸以下不建议，还是感觉比较小，可以考虑桌面摆件。建议大小10寸以上。
  实际选购自己做决定！大小因人而异，想要多大显示区域，就用多大的显示器。有闲置的可以利用起来，显示器有两种选择：1.用台式机显示屏2.用笔记本拆机屏

台式机显示屏

  优点是即插即用，不用另外配驱动板了，大屏幕会相对便宜点，缺点是比较厚重，外框要质量好，要厚。
  用台式机显示屏时看好是什么接口，HDMI的就直接用连接线连上树莓派的HDMI口就可以了，VGA的接口需要转换线，HDMI转VGA，买线的时候注意。
  屏幕选择时，接口的方向尽量是在侧面，这样挂在墙上好插线点

笔记本拆机屏

  用笔记本拆机屏的优点是比较薄，做出来相对薄，缺点是需要额外配驱动板，选择屏幕的时候，尽量不要选择下图这种边上有“耳朵“的。

  我以14寸为例，找一个尽量没有“耳朵”的，大概是下图这样子，屏幕只要型号不太旧就行

  屏幕的驱动要配一块驱动板，屏幕上有型号，我的是下图这样，然后在某宝上搜索“屏幕驱动板”，将型号发给他，他会配好驱动板，拆机屏+驱动板这一套差不多200左右。

原子镜

  原子镜是个什么东西呢？就是从暗的一面能看到亮的一面，从亮的一面看不到暗的一面，广泛应用于监狱、公检法机构审讯室、精神病医院、大学科研机构研究室、大型会议室等。这样显示屏上的光才能透出来，又能达到镜子的效果，这个就是淘宝找一家，挑便宜的就OK，有的卖到200+一块的就不要考虑了。
  如果家附近有玻璃厂，可直接去厂里，会便宜很多，因为这玩意贵的就是运费。

原子镜的替代品

  1. 玻璃上贴单透膜：优点： 成本降低，缺点：需要高超的贴膜技术，很容易有气泡和灰尘，表面容易有划痕，效果肯定没有原子镜好。
  2. 亚克力镜面板：即厂家在亚克力板出厂时候，就贴好了单透膜，机器无尘镀膜，不要买太薄的，不然镜面会扭曲的，同样，效果没有原子镜好。优点：降低成本，降低重量。缺点：表面容易后划痕，镜面容易弯曲，效果肯定比不上原子镜。 需要的话淘宝直接搜，成本能降低。

外框

  镜子的外框常见的有金属和木质两种。暂未用过金属的，所以这里暂只介绍木框选择。

  1. 要有挡板，能拖住镜子和显示器即可
  2. 背面要厚一点，要把树莓派塞进去呢，如下图，背面至少要有4cm以上的深度

固定与安装

  依次在木框里面放上:原子镜，显示屏，驱动板和树莓派，连接好就可以完美显示啦！但是这样子里面的部件没有固定，如果有木工基础，可以自己做卡扣，如果动手能力比较差，那就用最原始的方法吧。
  如果显示屏想重复利用(比如我平时用作电脑的副屏)，就简单用一些胶带在边缘贴住。如果说显示屏就固定不动了，可以考虑用AB胶，这可是超级牢固的，涂抹的时候避开裸露电线部分，此过程不可逆哟！
